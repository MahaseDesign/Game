Year 8 Computer Science

Topics Covered

  • Introduction to Programming
  • Understanding Algorithms
  • Basic Data Structures
  • Web Development Basics
  • Cybersecurity Fundamentals

Learning Outcomes

Students will be able to:

  • Write simple programs using a programming language.
  • Understand the importance of algorithms in computer science.
  • Identify common data structures and their uses.
  • Create a basic webpage using HTML and CSS.
  • Recognize common cybersecurity threats and how to protect against them.
